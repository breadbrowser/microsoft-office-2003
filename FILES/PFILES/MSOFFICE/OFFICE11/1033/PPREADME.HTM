Pack and Go has been replaced by Package for CD

The Pack and Go feature included with earlier versions of PowerPoint has been replaced with the Package for CD feature in Microsoft Office PowerPoint 2003. Package for CD allows you to save PowerPoint presentations and their supporting files to a file or a CD. You have the option to include the PowerPoint Viewer for playback on computers that are not running Office PowerPoint 2003.
