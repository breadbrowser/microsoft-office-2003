Upgrading issues

Grammar, thesaurus, and hyphenation .lex files may be missing after upgrading from Microsoft Office XP

If users who do not have the latest service packs installed for Microsoft Windows XP or Microsoft Windows 2000 (Windows XP SP1 or Windows 2000 SP4) upgrade from Microsoft Office XP to Microsoft Office 2003 and then access grammar features, the grammar, hyphenation, and thesaurus .lex files may be missing. For example, in the U.S. English version, these files are: grammar (Msgr3en.lex), hyphenation (Mshy2_en.lex), and thesaurus (Msth3am.lex, Msth3br.lex).

To prevent this problem, use the Spelling and Grammar command on the Tools menu to access grammar features. Using this command installs the necessary files.
